    Description

    The software contained in this repository is a set of Actor Critic Designs (ACDs) that are using reinforcement learning (RL) in order to solve the standard cart-pole problem.

    Our implementations include:

    • HDP
    • ADHDP
    • DHP
    • ADDHP
